Support

Get help with Quran MCP — an MCP server providing grounded Quran data to any AI client.

support@quran.ai

We typically respond within 48 hours.

What to include

Choose the category that best matches your issue.

Bug Report Describe what happened, what you expected, which AI client you're using (ChatGPT, Claude, etc.), and the tool that failed.
Feature Request Describe the capability you'd like and how it would help your use case.
Data Issue If you notice incorrect Quran text, translation, or tafsir content, let us know the ayah reference and edition.
General Question Anything else — setup help, edition availability, integration questions.

Other channels

GitHub Issues and a Discord community are planned for the future. For now, email is the primary support channel.
